## 📖 About The Project

**Friendly Dev** is a personal portfolio and blog website built as a hands-on learning project while exploring **React**, **React Router v7**, and a modern full-stack toolset. It showcases projects, shares blog posts, and lets visitors get in touch — all served through a clean, responsive interface powered by real-world tools like a headless CMS and a cloud database.

> 🎯 Built for **general users** — visitors, recruiters, and anyone curious about the developer behind the screen.

---

## ✨ Features

- 📝 **Dynamic Blog** — Blog posts managed via Strapi CMS and served from a Neon (PostgreSQL) cloud database
- 💼 **Portfolio Showcase** — Dedicated projects page to highlight work and skills
- 📬 **Working Contact Form** — Functional contact form powered by FormSpree — no backend needed
- ⚡ **Fast & Modern** — Built with Vite for lightning-fast dev experience and optimized builds
- 🎨 **Responsive Design** — Tailwind CSS ensures a polished look on all screen sizes
- 🔀 **Client-side Routing** — Smooth page navigation with React Router v7
